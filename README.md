# Pet-Management-Platform-node.js
# 宠物管理平台 EXpress+vue+MySQL
### 1、介绍
本项目围绕 宠物门店、宠物医院与会员服务 等核心业务进行设计，功能完整、流程清晰，主要分为管理后台+用户端前台两部分，使用express框架+vue+mysql前后端分离！
### 2、源码
地址：https://www.wwwoop.com/home/Index/projectInfo?goodsId=140&typeParam=1&subKey=-1

页面有演示站，可查看~

开发不易，小偿获取，谢谢理解~

### 3、功能菜单
① 管理后台功能

· 首页

· 会员管理

· 会员充值

· 宠物类型

· 宠物档案

· 疫苗记录

· 医疗记录

· 宠物用品

· 订单管理

· 索赔记录

· 管理员管理

② 用户端（前台）功能

· 会员登录/注册

 · 首页展品展示（宠物及用品展示、活动推荐等）

 · 购物车管理

 · 在线下单与订单查询

 · 个人中心（个人信息、我的宠物、订单历史等）

 ### 技术栈
 
 · 后端：Node.js 20.19.6 + Express 框架

· 数据库：MySQL 8.0

· 包管理 / 构建工具：npm

· 前端：Vue 2 + Element UI

· 开发工具：VSCode

系统既支持管理者高效运营，又为用户提供友好便捷的前台体验！
